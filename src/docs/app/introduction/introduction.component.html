SBB Lyne Angular Component Library

SBB’s digital guidelines ensure a consistent and recognizable user experience across all digital channels, helping customers quickly find the information they need while providing binding visual standards for all internal and external digital projects.

This library is being developed on GitHub ( sbb-design-systems/lyne-components and sbb-design-systems/lyne-angular ). Any contribution is always welcome. If you would like to report a problem or have a feature request, feel free to open an issue in either lyne-components or lyne-angular .

Packages @sbb-esta/lyne-angular Standard components that can be used for any kind of web site and web application. @sbb-esta/lyne-elements-experimental Experimental components that can change at any time. How to update
